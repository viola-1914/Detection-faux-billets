
💵 Détection de faux billets avec Machine Learning
🚀 Présentation

Ce projet a pour objectif de développer un modèle de Machine Learning capable de distinguer les billets authentiques des faux billets en se basant sur des données statistiques (dimensions, poids, caractéristiques visuelles, etc.).

L’application illustre comment les algorithmes d’apprentissage supervisé peuvent être utilisés dans un contexte de sécurité bancaire et financière.

🎯 Objectifs

Mettre en place un pipeline de Machine Learning complet :

Prétraitement et nettoyage des données

Analyse exploratoire des variables

Entraînement d’un modèle de classification

Évaluation des performances

Fournir un modèle capable de prédire si un billet est authentique ou faux.

⚙️ Fonctionnalités

Chargement et analyse du dataset des billets.

Visualisation statistique des variables pour identifier les différences entre vrais et faux billets.

Implémentation de plusieurs algorithmes de classification (par ex. Régression Logistique, Random Forest, SVM).

Évaluation des modèles à l’aide de métriques comme précision, rappel, f1-score et matrice de confusion.

Sélection du meilleur modèle pour la prédiction finale.

🛠️ Technologies utilisées

Python (langage principal)

Bibliothèques ML & Data Science : Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn

Outils : Jupyter Notebook, Git
